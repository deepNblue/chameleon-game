# 🦎 变色龙大冒险 (Chameleon Adventure)

> 一款基于Phaser 3的网页休闲益智游戏

---

## 📋 项目简介

**变色龙大冒险**是一款网页版休闲益智游戏，玩家控制变色龙根据环境颜色变化，躲避天敌，捕食昆虫。

### 核心玩法
- 🎨 **颜色匹配机制**: 根据背景颜色变化，躲避天敌
- 🐛 **捕食系统**: 捕食昆虫获得能量
- 🌍 **多场景关卡**: 热带雨林、沙漠、花园等
- ⚡ **能量系统**: 管理能量进行变色和移动

---

## 🎮 游戏特性

- 动态颜色系统
- 多种昆虫类型
- 多场景关卡
- 积分排行榜
- 能量和技能系统
- 音效和背景音乐

---

## 🛠️ 技术栈

- **游戏引擎**: Phaser 3
- **开发语言**: TypeScript
- **构建工具**: Vite
- **代码规范**: ESLint + Prettier

## 🚀 快速开始

### 安装依赖
```bash

```

### 开发模式
