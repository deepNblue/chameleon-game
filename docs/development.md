## 游戏设计

### 核心玩法
1. **颜色匹配机制**: 变色龙根据背景颜色变化，躲避天敌
2. **捕食系统**: 捕食昆虫获得能量和分数
3. **生存挑战**: 在时间内完成关卡，躲避天敌

### 操作方式
- **方向键**: 控制变色龙移动
- **空格键**: 变色（消耗能量）
- **ESC键**: 暂停游戏

### 游戏机制

#### 颜色匹配系统
- 使用欧几里得距离计算颜色相似度
- 匹配度范围：0-100%
- 隐身阈值：70%以上判定为隐身成功
- 隐身成功时天敌无法发现

#### 能量系统
- 初始能量：100
- 变色消耗：10能量
- 捕食奖励：15能量
- 能量不足时无法变色

#### 昆虫类型
| 类型 | 分数 | 速度 | 特点 |
|------|------|------|------|
| 🦋 蝴蝶 | 100 | 快速 | 高分值，难以捕捉 |
| 🐝 蜜蜂 | 50 | 中等 | 直线飞行 |
| 🦗 蟋蟀 | 30 | 慢速 | 容易捕捉 |

#### 天敌类型
| 类型 | 速度 | 伤害 | 特点 |
|------|------|------|------|
| 🐍 蛇 | 慢速 | 高 | 缓慢但致命 |
| 🦅 鸟 | 快速 | 中 | 快速俯冲 |

## 架构设计

### 目录结构
```
chameleon-game/
├── src/
│   ├── main.ts              # 入口文件
│   ├── scenes/              # 游戏场景
│   │   ├── BootScene.ts     # 加载场景
│   │   ├── MenuScene.ts     # 主菜单
│   │   ├── GameScene.ts     # 游戏主场景
│   │   ├── GameOverScene.ts # 游戏结束
│   │   └── PauseScene.ts    # 暂停场景
│   ├── entities/            # 游戏实体
│   │   ├── Chameleon.ts     # 变色龙
│   │   ├── Insect.ts        # 昆虫
│   │   └── Enemy.ts         # 天敌
│   ├── systems/             # 游戏系统
│   │   ├── ColorSystem.ts   # 颜色系统
│   │   └── EnergySystem.ts  # 能量系统
│   └── config/              # 配置文件
│       └── GameConfig.ts    # 游戏配置
├── assets/                  # 游戏资源
├── public/                  # 静态文件
└── docs/                    # 文档
```

### 场景流程
```
BootScene (加载)
    ↓
MenuScene (主菜单)
    ↓
GameScene (游戏) ←→ PauseScene (暂停)
    ↓
GameOverScene (游戏结束)
```

### 核心类设计

#### Chameleon类
- 属性：位置、颜色、能量
- 方法：move(), changeColor(), matchColor()
- 功能：变色龙控制和颜色匹配

#### Insect类
- 属性：类型、分数、速度
- 方法：update(), getScore()
- 功能：昆虫AI和移动

#### Enemy类
- 属性：类型、速度、伤害
- 方法：update(), patrol()
- 功能：敌人AI和巡逻

#### ColorSystem类
- 静态方法：
  - calculateMatch(): 计算颜色匹配度
  - lerpColor(): 颜色插值
  - isHidden(): 判断是否隐身

#### EnergySystem类
- 属性：energy, maxEnergy
- 方法：useEnergy(), addEnergy()
- 功能：能量管理
